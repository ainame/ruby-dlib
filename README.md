# Dlib

Ruby binding of [dlib C++ library](http://dlib.net/).

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'dlib'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install dlib
